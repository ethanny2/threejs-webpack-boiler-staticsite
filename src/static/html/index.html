Threejs webpack boilerplate

About

This project was created to provide a good starting point for learners who are new to webpack, javascript bundlers, and Threejs. I wanted to create an optimal development and production environment that was simple to understand and hassle-free to set up for experts and novices alike.

Features

webpack prod config set up with best practices for caching, minimization and babel transpliation to ES5

Through Github documentation that includes citations, free learning resources, a glossary and more

Draco Compression via the GlftPipeline package to compress .gltf models for best performance

Improved Three render loop syntax and examples using Tweenjs and Shader Particle Engine
